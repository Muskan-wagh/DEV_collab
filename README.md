# Developers Collaboration Platform

A full-stack developer collaboration platform built to help developers connect, collaborate, manage projects, and share ideas in real time. The platform provides a modern and scalable environment for teamwork, communication, and productivity.

---

## Features

- User Authentication & Authorization
- Developer Profiles & Team Collaboration
- Real-time Chat & Discussions
- Project & Task Management
- Create, Update, and Track Tasks
- Responsive Modern UI
- Fast and Scalable Architecture
- Smooth User Experience
- Notifications & Activity Updates

## Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/DEV_collab.git
```

### 2. Navigate to the Project Folder

```bash
cd DEV_collab
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Setup Environment Variables

Create a `.env` file in the root directory and add:

```env
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
```

### 5. Run the Development Server

```bash
npm run dev
```
